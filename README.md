# 후기

협업을 통해 프로젝트 기획에서부터 폴더 구조 설정, 번들링 도구, gitflow 사용 등 혼자서 공부할땐 미처 공부하지 못했던 부분들까지 공부하며 많이 배울 수 있어 너무 유익하고 좋았던 시간이었습니다. 리액트와 타입스크립트를 사용한 프로젝트는 이번이 처음이여서, 리액트와 타입스크립트가 익숙해진 시점에 다시 한번 코드들을 검토하며 리팩토링을 진행해 보고 싶습니다. 이번 프로젝트에서는 갤러리 파트를 담당하여 구현하였지만, 이후 리팩토링을 진행하며 다른 팀원들이 맡았던 마크다운, 로그인 페이지 등의 기능구현도 직접 해 볼 생각입니다. 이번 과제를 통해 리액트와 타입스크립트에 대해 더욱 열정을 불태울 수 있는 계기가 된 것 같습니다! 특히 이번 프로젝트에서 정말 좋았던 점은 다른 사람들과 의견을 주고 받으면서 여러 해결 방법에 대해 알 수 있었다는 점입니다. 개발에 대한 시야가 넓어진 느낌을 받아서 좋았습니다. 어렵고 모르는 부분들도 조원분들께서 친절히 알려주셔서 너무 너무 감사했습니다!!!! 😊
